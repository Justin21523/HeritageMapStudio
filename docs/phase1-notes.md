# Phase 1 Notes

## Goal

Build the first runnable desktop application shell for HeritageMap Studio.

## Important Decisions

- UI language: English
- Code comments: Chinese, concise and practical
- Platform: Linux
- IDE: VS Code
- Build: CMake
- GUI: Qt6 Widgets
- Rendering: QOpenGLWidget + OpenGL 3.3 Core Profile

## What is intentionally not included yet

- SQLite database
- CSV / JSON import
- real GIS projection
- map tile background
- timeline filtering
- route editing

These are planned for later phases.
